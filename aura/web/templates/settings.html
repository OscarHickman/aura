How It Works
  1. Fetch: Papers are downloaded from arXiv for your configured astrophysics categories.
  2. Embed: Each paper's title + abstract is converted to a 384-dimensional vector using a sentence-transformer model.
  3. Score: Your neural network preference model scores each paper (0-1) based on learned preferences.
  4. Rate: You thumbs-up or thumbs-down papers. Each rating immediately updates the model (online learning).
  5. Improve: The more you rate, the better the recommendations get. Use "Full Retrain" periodically for best results.
